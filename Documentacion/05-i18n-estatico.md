# 05 — Internacionalización en Astro (i18n)

## Estrategia

En el proyecto original (React + Django), el idioma se gestionaba con `react-i18next` en frontend y `django-parler` en backend. En la versión estática, toda la i18n vive en Astro:

- **Textos de UI**: archivos JSON en `src/i18n/`
- **Contenido de artículos**: archivos Markdown separados por carpeta de idioma
- **Rutas**: `/es/`, `/en/`, `/fr/` generadas por Astro i18n routing

---

## Configuración en astro.config.mjs

```javascript
export default defineConfig({
  i18n: {
    defaultLocale: 'es',
    locales: ['es', 'en', 'fr'],
    routing: {
      prefixDefaultLocale: true,  // /es/ también tiene prefijo
    },
    fallback: {
      en: 'es',   // Si no hay traducción EN, usa ES
      fr: 'es',   // Si no hay traducción FR, usa ES
    },
  },
});
```

Con `prefixDefaultLocale: true`, todas las rutas son consistentes:
- `/es/` — Español
- `/en/` — English
- `/fr/` — Français

## Helper para traducciones (src/i18n/utils.js)

```javascript
import es from './es.json';
import en from './en.json';
import fr from './fr.json';

const translations = { es, en, fr };

export function useTranslations(lang) {
  return function t(key) {
    const keys = key.split('.');
    let value = translations[lang] || translations['es'];
    for (const k of keys) {
      value = value?.[k];
    }
    return value || key;
  };
}
```

### Uso en una página Astro

```astro
---
import { useTranslations } from '../i18n/utils.js';
const lang = Astro.currentLocale || 'es';
const t = useTranslations(lang);
---

<h1>{t('home.latestArticles')}</h1>
<nav>
  <a href={`/${lang}/`}>{t('nav.home')}</a>
  <a href={`/${lang}/fiestas`}>{t('nav.fiestas')}</a>
</nav>
```

## Selector de idioma (componente)

```astro
---
// src/components/LanguageSwitcher.astro
const { currentLang } = Astro.props;
const currentPath = Astro.url.pathname;

// Reemplaza el prefijo de idioma en la URL actual
function getPathForLang(lang) {
  return currentPath.replace(/^\/(es|en|fr)/, `/${lang}`);
}
---

<div class="lang-switcher">
  <a href={getPathForLang('es')} class:list={['lang-btn', { active: currentLang === 'es' }]}>ES</a>
  <a href={getPathForLang('en')} class:list={['lang-btn', { active: currentLang === 'en' }]}>EN</a>
  <a href={getPathForLang('fr')} class:list={['lang-btn', { active: currentLang === 'fr' }]}>FR</a>
</div>
```

## Artículos multiidioma en Content Collections

Para leer artículos del idioma actual:

```astro
---
import { getCollection } from 'astro:content';

const lang = Astro.currentLocale || 'es';

// Lee todos los artículos del idioma actual
const articles = await getCollection('articles', (entry) => {
  return entry.id.startsWith(`${lang}/`);
});

// Ordena por fecha descendente
articles.sort((a, b) => b.data.publishedAt - a.data.publishedAt);
---
```

## Rutas dinámicas para artículos

```astro
---
// src/pages/[lang]/articulos/[slug].astro
import { getCollection } from 'astro:content';

export async function getStaticPaths() {
  const allArticles = await getCollection('articles');

  return allArticles.map((article) => {
    const [lang, ...rest] = article.id.split('/');
    return {
      params: {
        lang,
        slug: article.data.slug,
      },
      props: { article },
    };
  });
}

const { article } = Astro.props;
const { Content } = await article.render();
---

<Content />
```
